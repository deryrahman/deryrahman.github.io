Information Extraction

Information Extraction (IE) merupakan suatu proses untuk meng-ekstraksi informasi dari unstructured atau semi-structured menjadi structured yang dapat dibaca oleh mesin. Teks yang biasa kita baca, merupakan contoh dari unstructured, sedangkan mesin hanya dapat memproses informasi yang terstruktur. Oleh karena itu, IE ini penting untuk memisahkan antar entitas penting pada suatu teks tidak terstruktur beserta relasinya.

Untuk mendapatkan informasi terstruktur dari teks yang tidak terstruktur, hal pertama yang perlu diperhatikan adalah informasi target sebagai informasi terstruktur yang akan diekstrak. Kita tertarik hanya kepada informasi penting saja. Oleh karena itu perlu ditentukan, target data yang akan diekstrak.

Web Scraping

Web Scraping merupakan proses ekstraksi data dari website. Data tersebut disajikan oleh website sebagai HTML (semi-structured), kemudian oleh web scraper, data yang penting akan diekstrak untuk kemudian dimasukan kedalam database (structured). Pada search engine, web scraping dilakukan setiap crawling berjalan. Crawler menelusuri setiap halaman dokumen, kemudian scraper melakukan extraksi data penting dari halaman web tersebut.

Pada dasarnya HTML dibangun dari 4 tipe elemen : struktur dokumen, block, inline, dan elemen interaktif. Setiap versi HTML dibangun berdasarkan DTD (Document Type Definition) yang mengatur bagaimana elemen dikonstruksikan. Model yang paling banyak digunakan untuk struktur dokumen HTML adalah model pohon.

Setiap tag HTML direpresentasikan sebagai simpul. Nested tag merupakan anak dari simpul tersebut. Isi dari setiap simpul merupakan sebuah informasi yang kemudian akan diekstrak informasi yang penting untuk kemudian dimasukan kedalam database. Ekstraksi ini dapat dilakukan dengan menggunakan Regular Expression.

Example : Scraping using Python

Pada kali ini, saya mencoba untuk melakukan scraping sederhana dengan menggunakan python. Contoh website yang akan saya gunakan adalah 

http://econpy.pythonanywhere.com/ex/001.html

Jika kita lihat, dalam website tersebut terdapat tag div dengan atribut title="buyer-info" dan setiap buyer-info mempunyai data berupa nama buyer dan harga barang. Masing masing berada dalam tag div dengan atribut title="buyer-name" dan tag span dengan atribut class="item-price".

Secara sederhana, program python yang dibuat untuk mengekstraksi entitas nama dan entitas harga adalah sebagai berikut : 

from lxml import html
import requests

page = requests.get('http://econpy.pythonanywhere.com/ex/001.html')
tree = html.fromstring(page.content)

buyers = tree.xpath('//div[@title="buyer-name"]/text()')
prices = tree.xpath('//span[@class="item-price"]/text()')

print 'Buyers: ', buyers
print 'Prices: ', prices
